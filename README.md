# CNC Path Extractor

## Overview
CNC Path Extractor is a Python-based tool designed to extract and process CNC path data efficiently. This tool helps in analyzing and converting CNC paths for various applications.

## Environment Details
- **Python Version:** 3.8.10

```bash

```
